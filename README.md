# Telegram-бот для платной печати

## Описание проекта
Telegram-бот предоставляет удобный интерфейс для загрузки документов, настройки параметров печати и выбора точки печати. Сервис упрощает процесс печат, автоматизируя расчеты стоимости и управление заказами.

## Основной функционал
- **Загрузка документов**:
  - Поддерживаемые форматы: PDF, DOCX, PNG, JPG.
- **Настройка параметров печати**:
  - Черно-белая / Цветная печать.
  - Количество копий.
- **Расчет стоимости**:
  - Стоимость печати зависит от параметров, заданных в конфигурации (`config.py`).
- **Выбор точки печати**:
  - Список доступных мест печати .
- **Корзина**:
  - Просмотр загруженных документов, их параметров и стоимости.
  - Редактирование корзины.
- **Оформление заказа**:
- **Информация и помощь**:
  - Ответы на часто задаваемые вопросы (FAQ).

## Установка
### Требования
- Python 3.10 или выше
- Telegram Bot API Token

### Установка проекта
1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-repo/print-bot.git
    cd print-bot
    ```
2. Создайте виртуальное окружение и активируйте его:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/MacOS
    venv\Scripts\activate     # Windows
    ```

3. Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
Создайте файл .env и добавьте токен вашего Telegram-бота:
    ```bash
    BOT_TOKEN=your_bot_token
    ```
Запустите бота:
    ```bash
    python main.py
    ```
## Конфигурация

Конфигурация проекта задается в файле `config.py`. 

### Поддерживаемые форматы файлов:
Список форматов файлов, которые можно загрузить для печати:
```python
SUPPORTED_FORMATS = ['pdf', 'docx', 'png', 'jpg']
```

## Конфигурация
Конфигурация хранится в файле config.py:
Поддерживаемые форматы файлов:
```python
SUPPORTED_FORMATS = ['pdf', 'docx', 'png', 'jpg']
```
Тарифы печати:
```python
RATES = {
    "black_white": 5,  # руб. за страницу
    "color": 10,       # руб. за страницу
    "a3_multiplier": 1.2  # Наценка за формат A3
}
```
## Структура проекта
```bash

├── kb.py             # Описание кнопок для интерфейса
├── requirements.txt  # Зависимости
```
